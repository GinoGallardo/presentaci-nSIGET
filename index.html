SIGET

Sistema Inteligente de Gestión y Evaluación de Tráfico

React + FastAPI + SUMO

¿Qué es SIGET?

Aplicación web para consultar, editar y simular semáforos (TLS) de una red de tránsito modelada en SUMO.

Pensada para redes urbanas como Buenos Aires: gestionar programas de semáforos y evaluar el tráfico mediante simulaciones.

  • Consultar cruces por número
  • Editar fases y guardar en el archivo de red
  • Lanzar simulación SUMO centrada en el cruce

Tecnologías

Frontend
React 19, TypeScript, Vite 7, Tailwind CSS 4
Backend
Python 3, FastAPI
Simulación
SUMO (Eclipse SUMO)
Datos
XML (.net.xml, .rou.xml)

Cómo funciona

  1. Carga: El backend carga el .net.xml al iniciar
  2. Búsqueda: Usuario ingresa número → backend devuelve cruce y programa
  3. Edición: Cambios en fases → Guardar → actualiza XML + backup
  4. Simulación: Genera rutas temporales → abre SUMO GUI centrado en el cruce

API REST (Backend)

MétodoRutaDescripción
GET/tls/cruce/{numero}Obtener datos del cruce
GET/tls/{tl_id}Obtener programa TLS
POST/tls/{tl_id}/programActualizar programa
POST/tls/simulation/start/{numero}Iniciar simulación
